Place instructor or dance teacher photos in this folder.

Supported file types for normal browser use:
- .jpg
- .jpeg
- .png

Recommended approach:
- keep the main instructor image named `rohini-p-shenoy.jpg`
- if you want to use a different file name, update the `instructor.image`
  value in `src/data/mock.js`

Example paths you can use in `mock.js`:
- /assets/images/instructors/rohini-p-shenoy.jpg
- /assets/images/instructors/teacher-1.jpeg
- /assets/images/instructors/teacher-portrait.png
